# Recall

## When to use

Use this skill when the user asks you to remember past information, previous decisions, preferences, or project history.

## How to use

Call the memory_recall tool with a clear query describing what to retrieve. Use the retrieved memories to answer the user or guide next steps.

## Example

Query intent: "What did we decide about the database schema last week?"

Call memory_recall with query: "database schema decision last week"
